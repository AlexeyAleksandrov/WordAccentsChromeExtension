# Словарь ударений - Chrome Extension

Расширение для браузера Chrome, которое позволяет сохранять правильные ударения в словах и автоматически расставлять их на веб-страницах.

## 📋 Возможности

- ✅ Добавление слов в личный словарь ударений через контекстное меню
- ✅ Интерактивный выбор букв для ударения
- ✅ Автоматическая расстановка ударений на всех веб-страницах
- ✅ Умная адаптация стиля: подчёркивание → жирный → курсив
- ✅ Управление словарём через popup-окно
- ✅ Поддержка множественных ударений в составных словах
- ✅ Работа с динамическим контентом

## 🚀 Установка

### Способ 1: Загрузка неопубликованного расширения

1. Откройте Chrome и перейдите на страницу `chrome://extensions/`
2. Включите "Режим разработчика" в правом верхнем углу
3. Нажмите кнопку "Загрузить распакованное расширение"
4. Выберите папку с файлами расширения (`WordAccentsChromeExtension`)
5. Расширение будет установлено и готово к использованию

### Способ 2: Создание иконок (опционально)

Перед установкой создайте папку `icons/` и добавьте три PNG иконки:
- `icon16.png` (16x16 px)
- `icon48.png` (48x48 px)
- `icon128.png` (128x128 px)

Если иконок нет, расширение всё равно будет работать, но отобразится стандартная иконка Chrome.

## 📖 Как пользоваться

### Добавление слова в словарь

1. Выделите нужное слово на любой веб-странице
2. Кликните правой кнопкой мыши
3. Выберите "Добавить в словарь" в контекстном меню
4. В появившемся окне кликните по букве (или буквам), на которую нужно поставить ударение
5. Нажмите "Сохранить"

### Просмотр и управление словарём

1. Кликните на иконку расширения в панели инструментов Chrome
2. Откроется popup с полным списком сохранённых слов
3. Вы можете удалить любое слово, нажав кнопку "Удалить"
4. Кнопка "Очистить весь словарь" удалит все слова разом

### Как работает расстановка ударений

Расширение автоматически находит все слова из вашего словаря на веб-страницах и выделяет ударные буквы:

- **Обычный текст**: буква подчёркивается
- **Подчёркнутый текст**: буква выделяется жирным
- **Жирный текст**: буква выделяется курсивом

## 🔧 Технические детали

### Используемые API

- **Chrome Storage API**: для сохранения словаря
- **Context Menus API**: для добавления пункта в контекстное меню
- **Content Scripts**: для модификации контента страниц
- **Message Passing**: для коммуникации между компонентами

### Хранение данных

Словарь сохраняется в `chrome.storage.local` в формате:
```json
{
  "слово": [0, 3],  // индексы букв с ударением
  "другоеслово": [2]
}
```

## 💡 Примеры использования

### Пример 1: Простое слово
- Выделите слово "замок" (в значении "строение")
- Добавьте в словарь с ударением на "а": з**а́**мок
- Теперь на всех страницах "замок" будет отображаться с подчёркнутой буквой "а"

### Пример 2: Сложное слово
- Выделите "электростанция"
- Можете выбрать несколько ударных букв для составного слова
- Сохраните и проверьте результат

## 🐛 Известные ограничения

- Расширение работает только с текстовым контентом
- Не обрабатывает текст внутри `<script>`, `<style>` и `<noscript>` тегов
- Поиск слов регистронезависимый, но точное совпадение
- Для очень больших страниц возможна небольшая задержка при первой загрузке
