# Как создать иконки для расширения

## Быстрый способ: Использование онлайн-конвертера

1. Откройте файл `icon-template.svg` в любом браузере
2. Перейдите на сайт https://www.adobe.com/express/feature/image/resize или https://www.iloveimg.com/resize-image
3. Загрузите SVG файл и создайте три версии:
   - 16x16 пикселей → сохраните как `icon16.png`
   - 48x48 пикселей → сохраните как `icon48.png`
   - 128x128 пикселей → сохраните как `icon128.png`
4. Поместите все три PNG файла в эту папку (`icons/`)

## Альтернатива 1: Использование Inkscape (бесплатно)

1. Скачайте Inkscape: https://inkscape.org/
2. Откройте `icon-template.svg` в Inkscape
3. Экспортируйте три версии:
   - Файл → Экспортировать PNG
   - Установите размер 16x16, экспортируйте как `icon16.png`
   - Повторите для 48x48 и 128x128

## Альтернатива 2: Создание своих иконок

Вы можете создать собственные иконки в любом графическом редакторе:
- Adobe Photoshop
- GIMP (бесплатно)
- Figma (онлайн)
- Canva (онлайн)

### Требования к иконкам:
- Формат: PNG
- Прозрачный фон (опционально)
- Три размера: 16x16, 48x48, 128x128 пикселей
- Имена файлов: `icon16.png`, `icon48.png`, `icon128.png`

## Альтернатива 3: Использование эмодзи (самый простой способ)

Если у вас нет времени на создание иконок, расширение будет работать и без них.
Chrome просто покажет стандартную иконку для расширений в режиме разработчика.

## Идеи для дизайна иконки:

- 📚 Книга с символом ударения
- á Буква с ударением
- Ā Символ макрона
- ◌́ Комбинирующее ударение
- 📝 Документ с выделенной буквой

---

**Совет**: Для простоты можете использовать крупный эмодзи 📚 как иконку и сконвертировать его в PNG!
